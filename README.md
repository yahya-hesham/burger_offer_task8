## üöÄ Project Overview

This single-screen application focuses on a high-conversion UI/UX design, showcasing:

* **Promotional Banner:** Custom UI with clipped corners for a modern look.
* **Product Info:** Clear display of product title, star ratings, and category.
* **Interactive Quantity Selector:** Built using a reusable custom button widget for state management.
* **Dynamic Pricing:** Highlighting discounted pricing vs. original pricing.
* **Action-Oriented:** A prominent "Add to Cart" call-to-action (CTA).
* **Social Proof:** A dedicated review and feedback section.
